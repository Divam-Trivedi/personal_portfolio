Overview

This is the site you're on right now. It was built from scratch to serve as a professional showcase for robotics and engineering work, without relying on any frontend frameworks or CMS platforms. Every section, animation, and interaction is hand-coded.

Approach

Design System: A cohesive dark theme was designed using CSS custom properties for consistent color tokens across the entire site. The palette was chosen for readability and contrast while reflecting a technical, engineering aesthetic.

Animations: CSS keyframe animations and the Intersection Observer API are used for scroll-triggered fade-ins, a typing name effect on the hero, and smooth section transitions — all without JavaScript animation libraries.

Infinite Skill Carousel: The skills logo section uses pure CSS animations for three rows of logos scrolling in alternating directions, with mask gradients for seamless edge fades.

Contact Form: Formspree handles form submissions without requiring a backend, keeping the site fully static and cheaply deployable.

Deployment: The site is deployed and live at divamtrivedi.com with Google Analytics for traffic monitoring.
